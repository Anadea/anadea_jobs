We are looking for a highly skilled Senior Frontend Developer to join our team. You will\
be responsible for creating modern, high-performance web experiences for both our\
customer-facing sites and internal tools.\
\
**Responsibilities**\
● Develop modern, high-converting user interfaces using effective marketing and\
design techniques.\
● Implement the frontend for our public sites and internal tools, ensuring compliance\
with web standards and full cross-browser and mobile responsiveness.\
● Integrate frontend components with PHP-based environments (experience with Twig\
or similar templating engines is a plus).\
● Execute tests across multiple browsers and devices to ensure maximum\
performance and reliability.\
● Collaborate closely with Product, Marketing, Design, QA, and Operations teams.\
● Perform regular code reviews to maintain quality and consistency across the team.\
● Write clean, reusable, and efficient code, adhering to established best practices and\
design patterns.\
\
**Requirements**\
● 5+ years of professional experience in frontend development.\
● Strong proficiency in HTML5, CSS3, JavaScript (vanilla and jQuery), and Git.\
● Solid understanding of PHP environments and familiarity with Twig or similar\
templating systems.\
● Ability to code from scratch using modern JavaScript (ES6+).\
● In-depth knowledge of CSS methodologies (e.g., BEM) and preprocessors such as\
SASS.\
● Comfortable using package managers (Yarn, NPM, etc.).\
● A passion for clean code, design patterns, and best practices.\
● Experience with A/B test coding and comprehensive browser testing/debugging.\
● Strong analytical and problem-solving skills.\
● Conversational Spanish and Intermediate English communication skills are\
required.\
● Proactive, flexible, self-driven, and with an entrepreneurial mindset.\
● Demonstrated ability to work independently and take ownership of features and\
projects.\
● Solid handling of AI tools (such as Cursor), extensive experience in creating AI\
skills, and proven ability to design and implement AI agents independently. Able to\
lead AI-driven initiatives and integrate AI functionalities.\
\
**Desirable Experience**\
● Experience contributing to projects using React and/or TypeScript for scalable\
frontend solutions.

● Proven experience in mentoring junior developers and participating in code\
architecture discussions
