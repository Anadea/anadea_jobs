**Responsibilities**\
● Test Execution & Support: Execute manual and automated test cases to ensure\
software quality throughout the development lifecycle, focusing on functional and\
regression testing.\
● Bug Reporting: Identify, document, and track software defects with precision,\
providing clear steps to reproduce them and following up on their resolution.\
● Collaboration in Strategy: Assist the QA team in the creation of test plans and\
documentation, learning to identify risks and prioritize testing efforts based on project\
needs.\
● Automation Learning: Support the automation of manual test cases, learning to\
work within existing frameworks and contributing to the maintenance of automated\
suites.\
● Cross-functional Participation: Participate in meetings with Product, Design, and\
Engineering teams to understand requirements and ensure that quality criteria are\
met from the start.\
● Maintenance of Documentation: Keep test case repositories and technical\
documentation up to date to reflect changes in the product.\
● Device Testing: Assist in verifying consistency across different browsers (Chrome,\
Firefox, Safari) and mobile devices to ensure a seamless user experience.\
\
**Requirements**\
● 1–2 years of professional experience in quality assurance or software testing\
roles.\
● Technical Foundation: Basic understanding of web technologies (HTML, CSS) and\
how web applications work.\
● Tooling: Familiarity with bug-tracking tools like JIRA and basic knowledge of test\
management software (e.g., XRAY. TestRail, Zephyr).\
● API Basics: Interest or basic experience using tools like Postman for simple API\
requests.\
● Attention to Detail: Strong observational skills to spot UI inconsistencies and\
functional flaws.\
● Growth Mindset: High motivation to learn new technologies, testing methodologies,\
and automation tools.\
● Communication: Conversational Spanish and Intermediate English speaking skills\
(ability to understand documentation and participate in technical syncs).\
● Teamwork: A proactive and collaborative attitude, comfortable asking questions and\
seeking feedback to improve.

**Desirable Experience**\
● Initial Automation: Basic knowledge of JavaScript/Typescript or Python and an\
interest in learning frameworks like Selenium or Cypress.\
● Agile Exposure: Previous experience working in Scrum or Kanban environments.\
● Technical Curiosity: Personal projects or courses related to software development\
or advanced testing techniques.
