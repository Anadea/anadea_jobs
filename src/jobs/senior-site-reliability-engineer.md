managing GCP resources\

● Experience with monitoring and logging tools specific to GCP (e.g., Cloud Monitoring, Cloud Logging, Prometheus, Grafana)\
● Experience with some programming languages (preferably Typescript, JS, Go or Python)\
● Knowledge of version control systems (e.g., Git)\
● A strong preference for working in agile and dynamic environments\
● Ability to contribute to teamwork and effective collaboration across teams\
