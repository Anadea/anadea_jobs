• Ship to learn. Proactively instrument features, measure impact, seek user feedback, and iterate or kill based on what you discover. You treat metrics as a compass, not a report caro

• Collaborate with engineers, designers, and leadership to shape the product roadmap. You'll have a real voice in what gets built and why.

• Deep Python, TypeScript & React proficiency on the frontend (Hooks, Context API, Vite, Shadcn Ul) and backend (Node.js, Express, REST APls, PostgreSQL).
